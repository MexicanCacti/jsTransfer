File Sharing Platform with Node.js

Prerequisites
Ensure the following are installed on your system:
- Node.js
https://nodejs.org/

- MongoDB Community Server
https://www.mongodb.com/try/download/community

- npm

Clone Repo:
```
git clone <this url>
cd jstransfer
```

Install Deps:
```
npm install
```

Then start mongodb, make sure to replace the variable
```
mongoDBURL
```
In server.js with the url to your connection.

Then run
```
npm start
```
